NetOps Challenge Winner – Alex Walker

By Aniket DaptariJune 18, 2015Uncategorized
No Comments

We announced the first OpenContrail NetOps Challenge alongside the OpenStack Summit Vancouver last month. The purpose of this NetOps challenge was primarily to get the NetOps community to be made aware of some of the recent work in the synergy between OpenStack Nova, Neutron and Heat, OpenContrail as a platform for Cloud Network Automation and dynamic service chaining, and some leading virtualized network functions like Junipers vSRX.

We are proud to announce that Alex Walker, Director fromD2 Networks Limitedhas demonstrated the synergy brilliantly. Please watch the followingvideodocumenting his solution. To summarize it in short, Alex used OpenStack Heat to orchestrate a virtualized service instance of vSRX in an instantiation of a cloud in his lab that was managed using OpenContrail. The vSRX was deployed to implement a security function in a multi-tenant cloud to filter traffic origination from tenant workloads.

Thanks to all for participating! If you are looking for a similar event to participate, you can check out the latest NetOps Challenge. Finally, dont forget to watch Alexs solution to this iteration of the NetOps Challenge, and visit OpenContrail.org.
